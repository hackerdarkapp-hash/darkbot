# 🤖 NexusAI Dark Bot (@darkwapbot)

  بوت تيليجرام ذكي يعمل بـ Groq (llama-3.3-70b-versatile)

  ## المميزات
  - 💬 محادثة ذكية مع ذاكرة
  - 🖼️ تحليل الصور بالذكاء الاصطناعي
  - 📄 تحليل الملفات (كود، JSON، CSV...)
  - 🎭 شخصيات ديناميكية قابلة للتخصيص
  - ⚡ Keep-Alive 24/7 (ping كل 60 ثانية)
  - 🌐 دعم متعدد اللغات

  ## المتغيرات المطلوبة في Render
  ```
  TELEGRAM_BOT_TOKEN=...
  GROQ_API_KEY=...
  ADMIN_TELEGRAM_ID=...
  APP_URL=https://nexusai-darkbot.onrender.com
  ```

  ## ملاحظات
  - جميع الأسرار والمفاتيح تُضاف في Render Environment Variables فقط
  - لا يوجد أي مفتاح أو توكن داخل الكود
